Audio Reactive Visualizer – GitHub Pages Paket

Schritte:
1) Neues Repo auf GitHub anlegen (public), Name z.B. audio-visualizer.
2) ZIP entpacken und alle Dateien/Ordner hochladen.
3) Settings → Pages: Deploy from a branch, Branch main, Folder /(root), Save.
4) Den angezeigten Pages-Link nutzen; auf dem Handy per „Zum Homescreen hinzufügen“ installieren.
